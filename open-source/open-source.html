Open Source

The free sharing of designs can be dated back to before the first computer. One such example is Henry Ford, in the early 20th century, who revolutionised the motor vehicle industry to combat monopoly.[1]


There are numerous examples of open source creations designed and intended for use in education. An exmaple of an introductory programming environment for children and beginners is Scratch. The programming language is itself open source and the platform follows open source principles too, i.e. all user projects can be opened up and anyone can take code blocks from others to use in their own programs. Scratch is widely used and, being open source, anyone can contribute to improve the platform and make it more effective in teaching young people how to program.

All massively-known applications and websites are built on top of open source software. Frameworks like Node.js are maintained for free by charitable contributors and thousands of web servers across the world rely on this software. Similarly, Google maintains Android, the mobile OS, which any phone company can use and stylise for their phones.

The Internet is built on the HTTP protocol. The standard for it was developed by Sir Tim Berners-Lee who gave it away for free. Free, well-designed solutions to problems are rapidly adopted so open source is a great way to spread positive change.
